# Charlotte Williams Baptism Program

This folder is ready to upload directly to the root of the GitHub repository.

## Files

- `index.html`
- `styles.css`
- `hero.png`
- `charlotte-baptism-qr.png`
- `favicon.png`
- `README.md`

## Upload to GitHub

1. Open the `charlotte-baptism` repository.
2. Choose **Add file → Upload files**.
3. Upload all six files from this folder. Upload the files themselves, not the enclosing folder.
4. Click **Commit changes**.
5. Wait 1–3 minutes for GitHub Pages to publish.
6. Test the refreshed site at:

`https://williamsjared79.github.io/charlotte-baptism/?v=20260723-new-art`

The QR code points to the permanent address:

`https://williamsjared79.github.io/charlotte-baptism/`
